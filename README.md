
# Snake - Pygame

This is a basic functioning version of the arcade game Snake build using Pyton and Pygame.

## Installation

Install pygame with pip

```bash
  pip install pygame
```
To run the game
```bash
  python -u "path/to/main.py" or
  python3 -u "path/to/main.py"

```
